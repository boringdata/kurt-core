# Kurt Web UI

Quickstart:

1. Install node dependencies

```bash
cd web
npm install
```

2. Install Python dependencies for API (in your Python venv)

```bash
pip install -r ../requirements-web.txt
```

3. Start with the CLI from the repo root:

```bash
kurt serve
```

This will start the FastAPI backend at `http://localhost:8765` and the Vite dev server at `http://localhost:5173`.
