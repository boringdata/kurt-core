# Getting Started with ACME Products

Welcome to the ACME documentation!

## Quick Start

1. Install the ACME toolkit
2. Configure your environment
3. Run your first command

## Installation

```bash
pip install acme-toolkit
```

## Configuration

Set your API key:
```bash
export ACME_API_KEY=your-key-here
```

## First Steps

Run the hello world example:
```bash
acme hello
```
